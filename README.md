# Shop App Flutter

A shoe shopping app built with Flutter.

## What it does

- Browse shoes from different brands
- Filter by Nike, Adidas etc
- View product details and prices
- Select shoe sizes

## Running the app

```bash
git clone https://github.com/jatinsingh007/shop_app_flutter.git
cd shop_app_flutter
flutter pub get
flutter run
```

## Built with

- Flutter
- Material Design
